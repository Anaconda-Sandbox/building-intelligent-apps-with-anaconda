# Your First Intelligent App: Foundations of Intelligent Software Systems

What's the difference between a singular agent and an Intelligent application? An  `agent` is one component of an `autonomous system`, but with one agent and a scheduler you have `intelligent workflows`; but with considerations for data ingress and egress, data maintenance, user authentication and authorization, persistent storage, CI/CD and collaborative programming, etc; you're now building a software system.

`conda-pypi` allows you to access any package repository on PyPI. It's currently an experimental tool. For production worklows, we recommend using `Pixi,` which has `pip`, `uv`, `conda` interoperability.
