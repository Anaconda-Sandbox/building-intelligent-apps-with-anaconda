# Your First Agent: Building Intelligent Workflows

If you want to learn more about [Chat Apps]() and [AI architecture patters](), go to [source](). In this resource, we're building a single use tool.

There are two ways to build. You can use [Anaconda Agent Builder]() which allows you to have a conversation through a chat window in [Anaconda Desktop]() to build your first agent. The second option is to "Build Your Own Agent." 


## Primers

Check out the [Modern Conda README]() and [Environment Set Up 101 README]() to learn about best practices for setting up your environment. [Modern Conda]() is aimed as a reference guide. [Environment Set Up 101]() is aimed as a primer for new Python developers, or language switchers, trying to understand how to navigate Python environments and will get you Python, conda, and Jupyter on your machine.
